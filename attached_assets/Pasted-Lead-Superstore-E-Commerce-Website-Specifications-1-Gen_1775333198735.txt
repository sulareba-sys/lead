1. Serial Code (Cart Stage)
	•	Generate a unique serial code immediately when items are added to cart.
•	Display the serial code on the cart page.
•	The code must persist and never change throughout the entire process.

2. Order Flow Consistency
	•	The same serial code from the cart must be used for:
•	Checkout page
•	Final order record
•	Admin dashboard
•	WhatsApp message
•	Do NOT generate a new code at checkout.

⸻

3. Database Storage

Each order must store:
	•	Serial Code
•	Customer details
•	Ordered items
•	Total price
•	Date/time
•	Order status (see below)

⸻

4. Order Status System (IMPORTANT)

Each order must have a status field with options:
	•	Pending
•	Processing
•	Delivered
•	Returned
•	Cancelled

5. Admin Control
•	Admin must be able to:
•	Change order status manually

6. WhatsApp Integration
